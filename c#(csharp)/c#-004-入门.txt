for 循环语句
for (/*初始表达式*/;/*条件表达式*/;/*增量表达式*/)
{
    循环代码逻辑;
}
第一个 : 一般声明一个临时变量 ,用来计数
第二个 : 表明进入循环的条件 一个bool类型的结果
第三个 : 用第一个空中的变量 进行 自增减运算
注意 第一次进入循环时 才会调用 第一个空中的代码
     每一次进入循环之前 都会判断第二个  满足才会进入循环逻辑
     执行完循环逻辑才会执行第三个空中代码
for (int = 0 ; i<10 ; i++)
{
    Console.WriteLine(i);
}
